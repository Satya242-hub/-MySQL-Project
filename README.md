**E-COMMERCE USER BEHAVIOR ANALYSIS USING MYSQL**



**PROJECT OVERVIEW**

This project analyzes an e-commerce user behavior dataset using MySQL.



The dataset contains user events such as:

\- View

\- Add to Cart

\- Purchase



The data was cleaned, validated, and analyzed using SQL queries.





**DATASET COLUMNS**

\- Event\_Time

\- Event\_Type

\- Product\_ID

\- Category\_ID

\- Category\_Code

\- Brand

\- Price

\- User\_ID

\- User\_Session





**DATA CLEANING**

During import, the following issues were resolved:



\- Decimal column corrections

\- Null value verification

\- Column mismatch validation

\- Error Code 1366 (Incorrect decimal value)

&nbsp; '#VALUE!' entries in numeric fields



**SQL ANALYSIS PERFORMED**

\- Total Row Count

\- Null Value Check

\- Price Analysis (Min, Max, Avg)

\- Top Expensive Products

\- Event Type Distribution
